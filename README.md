# linux-lab-permissoes-escolares
Projeto de laboratório pessoal para estudo prático de Linux, com foco em gerenciamento de usuários, grupos e permissões de acesso.

O cenário simula um ambiente escolar, contendo alunos, professores, coordenação, administração e equipe de TI, cada um com níveis de acesso distintos a diretórios e arquivos sensíveis, como avaliações, notas e boletins.

O objetivo do projeto é compreender, na prática, como o Linux lida com controle de acesso, permissões, ownership e organização de usuários, sem o uso inicial de scripts ou automações.

Todo o versionamento deste repositório é voltado à documentação das decisões técnicas, estrutura lógica e evolução do laboratório.
A execução prática ocorre em uma máquina virtual Linux separada.

Tecnologias e conceitos abordados:
– Linux (usuários, grupos, permissões)
– Controle de acesso baseado em grupos
– Organização de sistemas multiusuário
– Documentação técnica
– Git e GitHub como apoio ao laboratório

Este projeto evoluirá gradualmente, partindo de configurações manuais para, futuramente, automações com scripts.
